# 📘 Axiom.io

A personal academic management system for students who refuse to guess.

Track grades, predict GPA, manage attendance, and see exactly where you stand — all in one place.

---

# 📌 Overview

Axiom.io is a modern academic dashboard designed to help students make data-driven decisions about their academic performance.

Instead of manually calculating grades or wondering whether attendance is safe, Axiom provides real-time insights into:

* Current academic standing
* GPA predictions
* Attendance tracking
* Subject-wise performance analysis
* Academic risk detection

This project combines clean UI/UX design, data visualization, and practical student productivity tools into a single intelligent platform.

# 🧠 Concept

Axiom.io is built around one core idea:

> Students perform better when they can clearly see where they stand.

Most students rely on scattered spreadsheets, rough calculations, or assumptions. Axiom centralizes academic data into a single system that provides clarity, prediction, and control.

It demonstrates both:

* Advanced frontend/dashboard development
* Real-world product thinking for student productivity

# ⚙️ Tech Stack

* **Charts & Analytics:** Recharts / Chart.js
* **State Management:** Zustand / Context API
* **Animations:** Framer Motion
* **Backend:** Firebase / Supabase *(optional)*
* **Deployment:** Vercel
